# Projet-RCP209 : Segmentation sémantique d'image

Implémentation de l'algorithme DeepLab dans le cadre du projet de l'UE [RCP209](https://cedric.cnam.fr/vertigo/cours/ml2/). L'algorithme à implémenter est issue de cet [article](https://arxiv.org/pdf/1606.00915.pdf)

Le jeu de données que nous utiliserons est PASCAL
